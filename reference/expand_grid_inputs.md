# Expand a grid of inputs

Expand a grid of inputs

## Usage

``` r
expand_grid_inputs(Input, ...)
```

## Arguments

- Input:

  A vector of inputs.

- ...:

  As many vector of covariates as desired. We advise to give explicit
  names when using the function.

## Value

A tibble containing all the combination of values of the parameters.

## Examples

``` r
TRUE
#> [1] TRUE
```
