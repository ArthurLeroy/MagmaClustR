# Weight follow-up data of children in Singapore

A subset of data from the GUSTO project (https://www.gusto.sg/)
collecting the weight over time of several children in Singapore. See
https://arxiv.org/abs/2011.07866 for dedicated description and analysis.

## Usage

``` r
weight
```

## Format

### `weight`

A data frame with 3,629 rows and 4 columns:

- ID:

  Indentifying number associated to each child

- sex:

  Biological gender

- Input:

  Age in months

- Output:

  Weight in kilograms
